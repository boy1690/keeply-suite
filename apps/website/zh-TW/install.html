安裝 Keeply
30 秒搞定。

個人免費。可離線使用。下面三條路任挑——不需帳號、不用註冊。

三條安裝路徑

挑適合你機器的，其他略過。

推薦 Windows · 30 秒

winget 一行指令

不會跳 SmartScreen 警告。Microsoft 的套件管理器幫你處理簽章審核。

winget install Boy1690.Keeply

開 PowerShell，貼上，按 Enter。

Windows · 5 分鐘

直接下載 .exe

不想開 PowerShell 的選這個。SmartScreen 會跳出來——點 「其他資訊」→「仍要執行」。就這樣。

下載 .exe

Windows 10 以上

macOS · 3 分鐘

.dmg 拖放安裝

首次開啟必須右鍵 → 打開（不是雙擊）才能繞過 Gatekeeper。只有第一次需要。

下載 .dmg

僅支援 Apple Silicon

Linux 用戶？還沒在 roadmap 上——Keeply 目前只出 Windows 與 macOS 版。

安裝完之後：丟進第一個專案

「裝完」不算結束，「同一天第一個專案被保護」才算結束。

  1. 1

    開啟 Keeply，點「新增專案」

    挑一個你正在用的資料夾——提案、合約、設計檔。不要挑一個半年沒動的封存資料夾。

  2. 2

    等 1–2 分鐘完成第一次掃描

    Keeply 對資料夾做一次索引。之後就在背景監看，每次儲存自動記錄版本。

  3. 3

    照常工作

    不需要按「儲存快照」按鈕。就跟平常一樣在你的編輯器按存檔。三天後要回溯，歷史紀錄都在。
