Cos'è un "Team Vault"?

Team Vault utilizza una topologia «1 Canonical + 1 Shared Mirror»: il Canonical è la fonte di verità per le versioni del progetto (NAS, cartella condivisa o disco esterno); il Shared Mirror è uno snapshot completo sincronizzato dal Canonical, in modo che qualsiasi membro del team possa ottenere rapidamente la versione più recente. Ciascun membro ha anche una propria copia di lavoro. Il formato dei dati può essere modalità Standard (sfogliare e modificare i file direttamente) o modalità Protetta (impedire modifiche distruttive alla cronologia).

Nessuna dipendenza dal cloud. Nessun costo per utente. I tuoi dati restano sul tuo hardware.

Domande frequenti

Keeply funziona offline? ▼
Sì. Keeply è software completamente locale — niente cloud, niente account, nessuna dipendenza dalla rete. I tuoi file di progetto, la cronologia delle versioni e le note restano tutti sul tuo disco rigido o NAS. Puoi usare Keeply in modalità aereo, in un seminterrato, o su una macchina completamente isolata dalla rete. L'unica volta in cui Keeply tocca la rete è quando controlla gli aggiornamenti dell'App, e puoi anche disattivarlo.
In cosa Keeply è diverso da Dropbox o Google Drive? ▼
Dropbox e Google Drive sincronizzano lo stato attuale dei file; Keeply registra ogni versione come uno snapshot separato. Dropbox e Drive fanno pagare per utente al mese; Keeply è gratis per i privati e avrà un Team plan a tariffa fissa di $25/month (in arrivo) indipendentemente dalla dimensione del team. I tuoi file vivono sul tuo disco, non sul server di un fornitore, quindi non sei legato a un abbonamento.
Cosa succede ai miei file se smetto di pagare? ▼
Niente. I file di Keeply sono salvati sul tuo disco rigido o NAS in una struttura di cartelle standard che controlli tu. Anche se disinstalli Keeply, i file e la loro cronologia delle versioni restano leggibili da qualsiasi file manager. Non stai pagando per accedere al tuo lavoro — la licenza Founding Member è un pagamento unico a vita, e il piano gratuito resta gratis per sempre.
Il mio team può condividere file senza licenze per postazione? ▼
Sì. Nel piano gratuito, i team condividono già una singola cartella vault via NAS o tramite un'unità condivisa — Keeply non conta le postazioni. Il prossimo Team plan ($25/month a tariffa fissa, previsto) aggiunge l'onboarding tramite codice invito e una dashboard amministrativa, ma sempre senza costi per postazione. Confronta con Dropbox Business a $20 per postazione al mese, che costa $120/month per un 6-person team.
Quali tipi di file supporta Keeply? ▼
Qualsiasi file. Keeply mostra un diff visivo affiancato per Word, Excel, PowerPoint, PDF e immagini. Per ogni altro tipo di file — Blender, PSD, video, CAD, modelli 3D, codice sorgente — Keeply tiene traccia delle versioni e ti permette di ripristinare qualsiasi punto in pochi secondi. I file oltre 10MB e 38 formati binari noti vengono rilevati in automatico e indirizzati allo storage per file di grandi dimensioni, senza alcuna configurazione né LFS da impostare.
Esiste una versione per Linux? ▼
Keeply è attualmente disponibile per Windows 10 o successivo e macOS su Apple Silicon. Una build Linux è nella roadmap ma non è ancora uscita. Se il supporto Linux è critico per il tuo flusso di lavoro, faccelo sapere tramite il modulo di contatto — diamo priorità alle piattaforme in base alla domanda reale degli utenti, e ogni richiesta viene registrata.
